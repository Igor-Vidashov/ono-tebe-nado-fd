# Оно тебе надо

Аукцион вещей, в которые никто не верил.

## О проекте

Одностраничный сайт, свёрстанный по макету. Проект выполнен в рамках обучения в Яндекс Практикуме.

## Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, позиционирование)
- Методология БЭМ

## Структура проекта
ono-tebe-nado-fd-main/
├── fonts/ # Шрифты Raleway
├── images/ # Изображения и иконки
├── styles/ # CSS-файлы
│ ├── fonts.css # Подключение шрифтов
│ ├── global.css # Глобальные стили и сбросы
│ └── style.css # Основные стили
└── index.html # Главная страница


## Как открыть

1. Скачайте или клонируйте репозиторий
2. Откройте файл `index.html` в браузере

## Особенности

- Полное соответствие макету
- Переиспользуемые компоненты (оверлей, адрес)
- Корректное отображение в Chrome, Firefox, Яндекс Браузере
- Валидный HTML и CSS
